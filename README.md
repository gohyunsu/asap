# ASAP English Slide Deck

English lab presentation deck for:

**ASAP: Aligning Simulation and Real-World Physics for Learning Agile Humanoid Whole-Body Skills**

- Paper: https://arxiv.org/abs/2502.01143
- Project: https://agile.human2humanoid.com/
- Code: https://github.com/LeCAR-Lab/ASAP
- Target GitHub Pages URL: https://gohyunsu.github.io/asap/

This is a static slide site. It uses a white, fixed-layout presentation style with 100 slides, hierarchical top navigation, keyboard navigation, hash links, and touch-friendly slide-body scrolling on mobile.

## Structure

1. Setup
2. Context
3. Foundations
4. Pre-training
5. Delta Action
6. Evidence
7. Synthesis

## Notes

The content is written as an English internal lab presentation rather than a literal translation of the Korean study deck. It removes explicit slide-role labels and keeps terms, formulas, derivations, related work, and empirical evidence inside the presentation flow.
