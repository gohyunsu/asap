# ASAP English Slide Deck

English lab presentation deck for:

**ASAP: Aligning Simulation and Real-World Physics for Learning Agile Humanoid Whole-Body Skills**

- Paper: https://arxiv.org/abs/2502.01143
- Project: https://agile.human2humanoid.com/
- Code: https://github.com/LeCAR-Lab/ASAP
- Target GitHub Pages URL: https://gohyunsu.github.io/asap/

This is a static slide site. It uses a white, fixed-layout presentation style with hierarchical top navigation, keyboard navigation, hash links, and deck-level scrolling so long slides remain readable without losing the fixed header.

## Structure

1. Setup
2. Transfer Gap
3. Pre-training
4. Alignment
5. Evidence
6. Takeaways

## Notes

The deck is intentionally compact. It keeps the fixed presentation format, but reduces repeated background material, sharpens each slide to one role, and keeps the paper's transfer loop easier to read from top to bottom.
