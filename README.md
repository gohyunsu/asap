# ASAP English Slide Deck

English lab presentation deck for:

**ASAP: Aligning Simulation and Real-World Physics for Learning Agile Humanoid Whole-Body Skills**

- Paper: https://arxiv.org/abs/2502.01143
- Project: https://agile.human2humanoid.com/
- Code: https://github.com/LeCAR-Lab/ASAP
- Target GitHub Pages URL: https://gohyunsu.github.io/asap/

This is a static slide site. It uses a white, fixed-layout presentation style with hierarchical top navigation, keyboard navigation, hash links, deck-level scrolling, and a compact final-presentation narrative structure.

## Structure

1. Setting
2. Landscape
3. Stage 1
4. Stage 2
5. Evidence
6. Takeaways

## Notes

The deck is intentionally compact and presentation-oriented. The current version is a 36-slide deck that follows the paper's own stage structure, uses paper figures/tables/metrics aggressively, and keeps presenter notes focused on natural Korean delivery plus concept explanations for newly introduced terms.
